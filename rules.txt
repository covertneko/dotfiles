## rsync rules for sync.sh
# Don't sync the following
- .git/
- .gitignore
- .gitmodules
- README.md
- sync.sh
- rules.txt
# If a directory is in mirror.txt, extraneous files will be deleted
- mirror.txt
# Don't delete the following when mirroring their parent directory
# Pp protects files unless their parent directory is removed(perishable)
# ** is needed to match all recursively
Pp .vim/backup/**
Pp .vim/undo/**
Pp .vim/swap/**
Pp .vim/bundle/*/**
